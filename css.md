Overview
--------
The elements of design:
* layout
* imagery
* color

* content 
* affordances
* typography


Simple example would be
```html
<html>
  
  <head>
  <style>
  h1 {text:bold;}
  #myid {color: green;}
  footer p {color: red; text: bold;}
  </style>
  </head>
  
<body>

  <h1> this should be bold </h1>
  <p id="myid"> this should be green </p>
  <footer> <p> This should be red and bold </p> </footer>

</body>
  
</html>
```
